第一章：Basic Structure of Computers

根据课件 Chapter 1 整理，围绕计算机类型、五大功能部件、程序执行和计算机世代展开。

  • Basic Structure of Computers
    • Computer Types
      • 分类依据size, cost, computational power, intended use
      • Embedded computers
        • 集成在更大的设备或系统中
        • 自动监测和控制物理过程或环境
        • 面向特定用途：工业/家庭自动化、家电、通信产品、车辆
      • Personal computers
        • 主要供个人专用
        • 支持通用计算、文档处理、CAD、影音娱乐等应用
        • 包括台式机、工作站、便携式和笔记本电脑
      • Server & Enterprise systems
        • 面向大量用户，通过公有或私有网络共享
        • 拥有大型数据库存储能力
        • 处理大量数据访问请求，用于企业和政府信息处理
      • Supercomputers & Grid computers
        • Supercomputers：最高性能，服务天气预报、工程设计、科学计算
        • Grid computers：把大量个人计算机和磁盘通过高速分布式网络组合
        • 中国超算示例：银河一号、曙光一号、天河一号/二号、神威太湖之光等
      • Cloud Computing：基于 Internet 按需提供共享资源、软件和信息
    • Five Basic Functional Units
      • 基本功能：Data Processing, Data Storage, Data Movement, Control
      • Input Unit：接收编码信息；设备包括键盘、触摸板、鼠标、操纵杆、轨迹球、相机、麦克风、网络通信设施等
      • Output Unit：把处理结果送到外部世界；设备包括打印机、图形显示器等
      • Memory
        • 存储程序和数据
        • Primary Storage：程序执行时所在的主存；关键词 Word, Word Length, Address, RAM, Memory Access Time
        • Cache Memory：更小更快的 RAM，保存当前执行的程序片段和所需数据
        • Secondary Storage：保存大量程序和不常访问数据，如磁盘、磁带、光盘、Flash
      • ALU：执行算术和逻辑运算，通常在处理器内完成大部分运算
      • Control Unit：协调其他部件，发送控制信号并感知各部件状态
      • 关系：I/O Unit 包括输入和输出；Processor 包括 ALU 和 Control Unit
    • Program Execution
      • 计算机处理的信息：instructions and data
      • Instruction：控制计算机内部以及与 I/O 设备之间的信息传送，并指定算术/逻辑操作
      • Program：完成任务的指令列表，存放在 memory 中
      • Instruction Cycle：Fetch Instruction + Execute Instruction；不断重复取指和执行
      • PC：指向下一条将被执行的指令
      • IR：保存当前正在执行的指令
      • 指令示例：Load R2, LOC；Store R4, LOC；Add R4, R2, R3
      • Interrupt：I/O 设备向处理器发出的服务请求；执行流程中需检查和处理中断
    • Computer Generations
      • Zero Generation：Mechanical Computers, 1642-1945；Pascal、Leibniz、Babbage、Zuse Z1
      • First Generation：Vacuum Tubes, 1945-1955；ENIAC、stored-program concept、汇编语言、延迟线/磁芯/磁鼓存储
      • Second Generation：Transistors, 1955-1965；晶体管、磁芯和磁鼓广泛使用、磁盘、高级语言 Fortran、编译器
      • Third Generation：Integrated Circuits, 1965-1975；IC、集成电路存储器、微程序、并行、流水线、cache、virtual memory、IBM360、PDP-11
      • Fourth Generation：LSI & VLSI, 1975-；单芯片处理器/主存片段、个人电脑、嵌入式、笔记本、移动电话、FPGA、层次化存储
      • 性能演进：真空管、晶体管、LSI/VLSI 带来数量级速度提升；体系结构上依赖 cache、pipelining 等
